The main class for the generation of passengers. 

Calls to \mbox{\hyperlink{classDPassengerFactory_a4ac00d43518f86c30798c58e719855f1}{Generate}} function to get a new instance of a passenger. This is a static call, not requiring an instance to invoke the method.
